U can test it for yourself, all you have to do is:
1. Copy repository to your disk.
2. Run `npm install` command at location of the ".../Unbored_App/" project, it will install all necessary dependencies.
3. Run `npm start` command at location of the ".../Unbored_App/" project, to run a virtual server with the site on your local network.
4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

   And you can test it for yourself, have fun! :)
